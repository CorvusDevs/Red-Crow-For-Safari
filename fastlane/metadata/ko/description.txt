Red Crow는 50개 이상의 맞춤형 기능으로 Safari에서의 YouTube 경험을 혁신합니다. 재생 속도, 화질, 볼륨 등을 브라우저를 떠나지 않고 세밀하게 조정할 수 있습니다.

사용 방법
확장 프로그램을 설치하고 YouTube를 열면 최적의 기본 설정으로 자동 작동합니다. 설정 페이지에서 모든 세부 사항을 맞춤 설정하거나, 빠른 프리셋으로 인기 있는 구성을 한 번의 클릭으로 적용하세요.

주요 기능

- 맞춤 재생 속도 - 0.25배에서 16배까지 기본 속도 설정, 채널별 기억 기능 포함.
- 화질 제어 - 기본 화질과 전체 화면 화질을 개별 설정.
- 볼륨 부스트 - YouTube 제한을 넘어 최대 600%까지 오디오 증폭.
- 시어터 & 시네마 모드 - 시어터 모드 자동 전환, 배경 어둡게, 완전 시네마 모드.
- SponsorBlock - 스폰서, 인트로, 아웃트로 등 비콘텐츠 구간 자동 건너뛰기.
- DeArrow - 클릭베이트 제목과 썸네일을 커뮤니티 제공 대안으로 교체.
- 방해 요소 숨기기 - 댓글, 관련 동영상, 쇼츠, 엔드 카드, 알림 등 제거.
- 맞춤 테마 - Nord, Solarized Dark, Catppuccin Mocha 중 선택하거나 YouTube의 라이트/다크 테마 강제 적용.
- 플레이어 도구 모음 - 스크린샷, 반복, A-B 반복, PIP, 역재생 등을 플레이어 내에서 조작.
- 동영상 필터 - 밝기, 대비, 채도, 블러, 색조 등을 실시간 조정.
- 스크롤 볼륨 - 플레이어 위에서 스크롤하여 볼륨 변경.
- 챕터 내비게이션 - 모든 챕터를 표시하는 슬라이드 패널로 빠른 탐색.
- 채널 프로필 - 속도, 화질, 볼륨, 시네마 모드의 채널별 설정 저장.
- 시청 타이머 - 설정한 시청 시간 후 휴식 알림.
- 키보드 단축키 - 반복, 스크린샷, PiP, 시네마 등의 맞춤 단축키.
- 미니 플레이어 - 페이지를 스크롤해도 따라오는 플로팅 미니 플레이어.
- 채널 차단 - 보고 싶지 않은 채널의 동영상 필터링.
- 길이 필터 - 원하는 범위보다 짧거나 긴 동영상 숨기기.
- 프리셋 - 팟캐스트, 프레젠테이션, 음악, 미니멀, 집중 모드 원클릭 설정.
- 맞춤 CSS - 자신만의 스타일을 삽입하여 완전한 시각적 제어.

설정 및 맞춤화
설정 페이지는 검색 바와 설명이 있는 명확한 섹션으로 구성되어 있습니다. 설정은 언제든지 가져오기, 내보내기, 초기화가 가능합니다.

개인정보 보호 우선
Red Crow는 개인 데이터를 수집, 저장 또는 전송하지 않습니다. 분석, 추적, 계정 없이 완전히 기기에서만 작동합니다. 브라우징은 비공개로 유지됩니다.

Safari를 위한 가장 포괄적인 YouTube 향상 확장 프로그램.
