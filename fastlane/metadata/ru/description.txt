Red Crow преображает ваш YouTube в Safari с помощью более 50 настраиваемых функций. Точная настройка скорости воспроизведения, качества, громкости и многого другого — всё без выхода из браузера.

КАК ЭТО РАБОТАЕТ
Установите расширение, откройте YouTube, и всё заработает автоматически с разумными настройками по умолчанию. Откройте страницу настроек, чтобы настроить каждую деталь, или используйте быстрые пресеты для применения популярных конфигураций одним щелчком.

ОСНОВНЫЕ ФУНКЦИИ

- Настройка скорости — Установите любую скорость по умолчанию от 0,25x до 16x с памятью для каждого канала.
- Управление качеством — Принудительное качество по умолчанию и отдельное качество для полноэкранного режима.
- Усиление громкости — Усильте звук до 600% сверх лимита YouTube.
- Театральный и кинорежим — Автоматический вход в театральный режим, затемнение фона страницы или полноценный кинематографический режим.
- SponsorBlock — Автоматический пропуск спонсоров, заставок, концовок и других нетематических сегментов.
- DeArrow — Замена кликбейтных заголовков и превью на альтернативы от сообщества.
- Скрытие отвлекающих элементов — Удаление комментариев, похожих видео, Shorts, конечных заставок, уведомлений и прочего.
- Пользовательские темы — Выбирайте из Nord, Solarized Dark, Catppuccin Mocha или принудительно установите собственную светлую/тёмную тему YouTube.
- Панель инструментов плеера — Скриншот, зацикливание, A-B-зацикливание, «картинка в картинке», обратное воспроизведение и многое другое — всё прямо в плеере.
- Видеофильтры — Регулируйте яркость, контраст, насыщенность, размытие, оттенок и другие параметры в реальном времени.
- Громкость прокруткой — Изменяйте громкость прокруткой над плеером.
- Навигация по главам — Выдвижная панель со всеми главами для быстрой перемотки.
- Профили каналов — Сохраняйте настройки для каждого канала: скорость, качество, громкость и кинорежим.
- Таймер использования — Напоминание о перерыве после настраиваемого времени просмотра.
- Горячие клавиши — Настраиваемые сочетания для зацикливания, скриншота, PiP, кинорежима и многого другого.
- Мини-плеер — Плавающий мини-плеер, который следует за вами при прокрутке страницы.
- Блокировка каналов — Фильтрация видео с каналов, которые вы не хотите видеть.
- Фильтр по длительности — Скрытие видео короче или длиннее указанного диапазона.
- Пресеты — Конфигурации одним щелчком для режимов Подкаст, Презентация, Музыка, Минимальный и Фокус.
- Пользовательский CSS — Внедрите свои собственные стили для полного визуального контроля.

НАСТРОЙКИ И ПЕРСОНАЛИЗАЦИЯ
Полная страница настроек организована в чёткие разделы с поисковой строкой и описаниями параметров. Импортируйте, экспортируйте или сбросьте свою конфигурацию в любое время.
