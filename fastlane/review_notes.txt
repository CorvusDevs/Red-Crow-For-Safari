Red Crow is a Safari Web Extension that enhances the video watching experience on youtube.com. It does not block ads or modify video streams — it adds user-facing controls and UI customizations on top of the existing web page.

HOW TO TEST
1. Open Safari and enable the extension in Safari > Settings > Extensions.
2. Navigate to youtube.com.
3. Click the Red Crow toolbar icon to see quick controls (speed, quality, volume, cinema mode).
4. Click "All Settings" in the popup to open the full settings page with 50+ options organized by category.

KEY FEATURES TO VERIFY
- Playback controls: Change default speed (0.25x–16x) and quality (144p–8K).
- Cinema Mode: Dims the page background to focus on the video player. Toggle via the popup or keyboard shortcut (Alt+Shift+C).
- SponsorBlock integration: Connects to the public SponsorBlock API (sponsor.ajay.app) to skip community-reported sponsor segments.
- DeArrow integration: Connects to the public DeArrow API to replace clickbait titles/thumbnails with community submissions.
- Hide Elements: Users can hide comments, related videos, Shorts, end cards, and other UI elements they find distracting.
- Video Filters: Adjust brightness, contrast, saturation, and other CSS filters on the video element.
- Channel Profiles: Save per-channel settings for speed, quality, and volume.
- Themes: Choose from 11 built-in color themes applied via CSS.
- Custom CSS: Users can inject their own CSS for further customization.

NETWORK REQUESTS
The extension makes requests to these third-party APIs only when the user enables the corresponding features:
- sponsor.ajay.app (SponsorBlock — segment data)
- returnyoutubedislikeapi.com (dislike counts)
- dearrow-thumb.ajay.app / sponsor.ajay.app (DeArrow — titles and thumbnails)
- 7tv.io, api.betterttv.net, api.frankerfacez.com (chat emotes for live streams)

No user data is collected, stored, or transmitted. No analytics. No accounts required.

PERMISSIONS
- storage: Save user preferences locally.
- activeTab: Apply settings to the current youtube.com tab.
- Host permission (youtube.com only): Inject content scripts for UI enhancements.
